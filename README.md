# React burger

Сайт космической бургерной. Можно собирать бургеры, создавать заказы, отслеживать текущее состояние заказа. 


Ссылка на проект: https://stellarburger.nomoredomainsclub.ru/

## Технологии

- typescriot
- react
- redux
- react-dnd
- websocket
- cypress


## Установка
```bash
npm i
```

## Запуск

```bash
npm run start
```

## Тесты

```bash
npm run test 
```

e2e тесты:
```bash
npx cypress open
```

## Деплой

```bash
npm run deploy
```
